# ElasticSearch + Logstash + Kibana (ELK Stack)
是一款強大的文本搜尋工具可以應用於:

- 網站搜索
- 日誌分析
- 地理空間分析處理等
- 其他

功能:
1. ElasticSearch: 搜尋引擎
2. Logstash: log 資料收集
3. Kibana: 資料視覺化
## 安裝方式

```base
cd elk; # 進入 elk 目錄
docker-compose up -d;
```
